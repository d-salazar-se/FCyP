# lab-progra

Programa que dibuja en la pantalla un arbol binario ordenado basado en una cadena de caracteres ingresada por el usuario.

Una vez iniciado el programa

python proyecto.py

Se pedira que ingreses una cadena de caracteres, en esta, no deben existir caracteres repetidos, ni espacios y solo se permiten las letras del alfabeto ingles (sin tildes, Ã±'s, etc.), de no cumplirse algunas de estas reglas se removeran estos caracteres invalidos y se generara una cadena nueva con los caracteres que si cumplan la regla. De ser una cadena vacia la resultante se termina la ejecucion del programa, en caso de haber al menos un caracter se muestra el arbol binario por pantalla y luego termina el programa.
